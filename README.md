
# 00_Dotfiles

A repository for development related environment and config files

## About

Contains various config files, `paths.json` in folders are used to automatically deploy files to their locations.  

Example paths.json  

```json
[
  {
    "files": [
      ".editorconfig"
    ],
    "paths": [
      "~/.atom"
      "/data/projects"
    ]
  }
]
```
